## Installation

1. Clone the repository.
```bash
git clone git@github.com:colbyrobins/pwa-text-editor.git
```
2. Install dependencies

```bash
cd pwa-text-editor

npm run install
```

3. Run the Application

```bash
npm start
```

4. Launch Browser
 - Navigate to https:http://localhost:3000/
